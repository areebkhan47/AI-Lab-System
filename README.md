# AI Lab Management System

## 📌 Project Overview
This project is a web-based (and later desktop-based) system designed to manage lab tasks for students and teachers. It includes separate portals for students and teachers, an integrated coding environment, and an AI-based automatic grading system.

---

## 🚀 Features

### 👨‍🏫 Teacher Portal
- Create, update, and delete lab tasks
- View student submissions
- Monitor performance
- AI-based grading and feedback

### 👨‍🎓 Student Portal
- View assigned lab tasks
- Write and submit code داخل system
- Get automatic grades and feedback
- Limited access to ensure focus during tasks

---

## 🤖 AI Integration
- Automatic grading of lab tasks
- Feedback generation for students
- Error detection and suggestions

---

## 💻 Built With

- Node.js (Backend)
- Express.js (API)
- MongoDB (Database)
- AI API (for grading)
- (Future) React / Electron for frontend

---

## 🎯 Objective
To create a fully dynamic system that allows teachers to manage lab tasks efficiently and students to complete and submit their work in a controlled environment.

---

## 📈 Future Enhancements
- Integrated code editor (IDE)

- Advanced AI grading system
- Real-time monitoring during lab tasks
